
**Project Title**
NaijaMart Sales Data Analysis

Project Overview
This project analyzes retail sales performance for NaijaMart using SQL, Excel, and data cleaning techniques.

The goal of this project was to clean messy retail sales data, analyze customer transactions, and build an interactive dashboard that provides business insights for better decision-making.

Business Problems Solved:

Identify top-performing product categories

Determine highest revenue-generating branch

Analyze payment preferences

Track monthly sales trends

Improve retail decision making through dashboard reporting

Tools Used
Microsoft Excel

SQL

Data Cleaning

Data Visualization

Dataset Information
The dataset contains:

Transaction IDs

Sales dates

Customer names

Product categories

Transaction amounts

Branch locations

Payment methods

Total Transactions After Cleaning: 289

Data Cleaning Process (SQL)
The raw dataset contained several issues:

Missing customer names

Duplicate transaction IDs

Negative sales values

Inconsistent date formats

Inconsistent branch names

Inconsistent payment methods

Category naming issues

Cleaning Steps Performed
Removed duplicates

Replaced missing customer names with "Unknown Customer"

Converted negative sales values to positive values

Standardized dates

Standardized branch names

Standardized payment methods

Standardized product categories

Key Findings
1. Total Revenue Generated
₦9,694,374.61

NaijaMart generated nearly ₦9.7 million in total sales revenue.

2. Best Performing Product Category
Household Products generated the highest revenue:

₦2,121,029.57

This shows strong customer demand for household products.

3. Second Best Category
Snacks generated:

₦1,321,161.94

This indicates fast-moving consumer goods perform strongly.

4. Highest Revenue Branch
Lagos Branch generated the highest revenue:

₦6,599,165.79

Lagos significantly outperformed:

Abuja → ₦1,717,226.94

Ibadan → ₦755,810.51

Kano → ₦622,171.37

5. Most Preferred Payment Method
Cash Payments led with:

₦3,625,977.67

Followed by:

Transfer → ₦3,093,982.04

POS → ₦2,974,414.90

This shows many customers still prefer cash transactions.

6. Peak Sales Month
January 2025 recorded the highest monthly sales:

₦848,282.51

This may reflect post-holiday consumer spending.

Business Recommendations
Increase Household Inventory
Since household products generate the most revenue.

Focus More Investment in Lagos
Lagos is the highest-performing branch.

Improve Digital Payments
Encourage POS and transfers to reduce cash handling risks.

Seasonal Promotions
Replicate January sales strategies during slower months.
